2. Выяснить, для чего предназначена команда cat. Используя данную команду, создайте два файла с данными, 
а затем объедините их в один. Просмотрите содержимое созданного файла. Переименуйте файл, дав ему новое имя.

Команду cat в большинстве случаев используют только для просмотра файлов. На самом деле одно из ее предназначений это конкатенация (объединение) файлов. 
А название команды происходит от слова catenate (сцепить), то есть присоединение одного после другого.
Под объединением файлов понимается их соединение.

sudo touch file.txt
sudo touch file2.txt
cat file.txt file2.txt > file12.txt
mv file12.txt file000.txt

4. В ОС Linux скрытыми файлами считаются те, имена которых начинаются с символа “.”. Сколько скрытых файлов в вашем домашнем каталоге? 
(Использовать конвейер. Подсказка: для подсчета количества строк можно использовать wc).

printf '%s\n' .[!.]* | wc -l

6. Запустить в одном терминале программу, в другом терминале посмотреть PID процесса и остановить с помощью kill, 
посылая разные типы сигналов. Что происходит?

Запустить в одном терминале программу ( man man), в другом терминале посмотреть PID процесса и остановить с помощью kill 
(смотреть с помощью htop и  ps aux | grep man ) ( kill -9 PID процесса), посылая разные типы сигналов. 
Что происходит. ( командой kill можно приостанавливать процесс возобновлять, завершать в штатном режиме и завершать критично).
